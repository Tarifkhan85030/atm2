ATM Simulator with Streamlit
This project is an interactive ATM (Automated Teller Machine) simulator created with Python and Streamlit. The application allows users to perform standard ATM functions such as generating a PIN, changing the PIN, checking their account balance, and withdrawing funds, all through a simple, web-based interface.

Features
PIN Generation: Set up an initial PIN for the account.
PIN Change: Securely update the PIN after verification.
Balance Inquiry: Check the current account balance.
Withdraw Funds: Withdraw a specified amount with balance validation.
Streamlit Interface: User-friendly web app with a clean, interactive interface.
Technologies Used
Python: Core language used for ATM logic.
Streamlit: Framework for creating an interactive web application.

Usage
Run streamlit run atm_simulator.py from your terminal to start the app.
Open the provided local URL in a web browser.
Follow the on-screen options:
Generate a PIN: Set up your initial PIN.
Change PIN: Update the PIN after entering the current one.
Check Balance: View the current account balance.
Withdraw Funds: Enter an amount to withdraw; balance updates accordingly.
